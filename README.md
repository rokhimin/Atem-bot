# Atem
discord & telegram bot for search yugioh card . written in Ruby (discordrb) .

### Play
> [Discord](https://discordapp.com/api/oauth2/authorize?client_id=617492380710469763&permissions=1074789568&scope=bot) | Telegram(soon..)

### use
###### discord 
- info :
> atem:info
- search card :
> ::name_card::

### developer tool tasks
- install gem :
> rake gem:install
- create db :
> rake db:create
- migration db :
> rake db:migrate
- drop db :
> rake db:drop
- test :
> rake run:rspec
- run bot discord :
> rake run:discord
- run bot telegram :
> rake run:telegram
